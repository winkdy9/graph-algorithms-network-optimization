# graph-algorithms-network-optimization
Реализация системы поиска кратчайших маршрутов на основе графового представления городской дорожной сети

# Поиск кратчайшего пути в городской дорожной сети

## О проекте

Проект представляет собой реализацию алгоритма поиска кратчайшего пути в графе на языке C с применением к реальным данным улично-дорожной сети Санкт-Петербурга.

Основная цель проекта — разработать собственное представление графа, реализовать загрузку данных и применить алгоритм Дейкстры для поиска оптимального маршрута между двумя точками.

## Реализовано

* проектирование структуры графа;
* загрузка графа из CSV-файлов;
* поиск ближайших узлов по координатам;
* алгоритм Дейкстры;
* восстановление найденного маршрута;
* взаимодействие C-программы с Python-скриптом для работы с адресами.

## Данные

Используются реальные данные дорожной сети:

* nodes.csv — вершины графа с координатами;
* edges.csv — рёбра графа с длиной и дополнительной информацией.

Каждая вершина представляет точку дорожной сети, а рёбра — соединения между участками дорог.

## Структуры данных

Для хранения графа использованы собственные реализации структур данных.

Использована самостоятельно реализованная структура динамического массива (vector), разработанная в предыдущих проектах.

## Алгоритмы

Реализован алгоритм Дейкстры для поиска кратчайшего пути во взвешенном графе.

Сложность алгоритма зависит от выбранного представления графа и структуры хранения соседей.

## Особенности реализации

* язык C;
* ручная работа с памятью;
* собственный парсинг CSV;
* отсутствие использования готовых библиотек графов;
* собственная реализация основных компонентов.
